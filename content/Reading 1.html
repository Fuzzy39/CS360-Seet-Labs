Arrays

What is an Array?

Arrays are a simple way to store several pieces of data. They allow fast random access through indexing. Any primitive type or object can be held in an array. Arrays have a length and are immutable which means that their size cannot be changed.

Declaring an Array

Arrays can be created by first declaring the type of value in the array followed by square brackets [] then the name of the array:

int[] myArray

Aternately place the square brackets after the array name:

int myArray[]

In these cases the array is named myArray and can store integers.

Instantiating an Array

There are several ways to instantiate an array. One way is done in the same line as the array is declared and requires all the values to be known beforehand. To do this use curly braces and separate the entries with commas:

int[] myArray = {1 3 5 7 9};

This creates an array with the same size as the number of entries listed.
Another way allows the size of the array to be set without populating the array. To do this use the new keyword and again use the type of value followed by square brackets. This time put the length of the array (the number of entries it should store) inside the brackets. It should look like this: 

int[] myArray = new int[10];

In this case myArray can store 10 integer values.

Accessing values in an Array

To access a value within an array use its Index. Arrays are zero-indexed which means that the first element in the array has an index of 0. The second value therefore has an index of 1 and so on. To access a value from an array by index use square brackets with the index inside:

myArray[0];

In this example the first entry will be retrieved.

Looping through an Array

Traversing arrays is an obvious use of loops. For example this is how to use a for loop to print all the values in an array:

for (int i = 0; i < myArray.length; i++) {
	System.out.println(myArray[i]);
}

Notice the use of the .length property which returns the number of elements which can be held in the array. This is also a way to get the last element in an array:

myArray[myArray.length - 1];

Also note that the index of the last item is length - 1 again because of zero-indexing. If you try to access an index which is outside the array an IndexOutOfBoundsException will be thrown.
