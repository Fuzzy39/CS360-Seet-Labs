Coding 5: Practicing OOP

Practicing OOP: Inheritance and Interfaces

In this activity, you're going to write a small hierarchy of classes to demonstrate your understanding of inheritance, interfaces, and method overriding. You'll define an interface called Animal, an abstract class called Mammal that implements the interface, and two concrete classes: Dog and Cat. Each class should have its own implementation of the makeSound() method. Finally, write a method called runTests() that creates a Dog and a Cat and returns their sounds concatenated together.
